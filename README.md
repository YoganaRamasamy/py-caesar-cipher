# Caesar Cipher Implementation 🔐

This project is part of my internship task.

## 📌 Task Objective
Create a Python program to encrypt and decrypt text using Caesar Cipher.

## ⚙️ Features
- Encrypts text using shift value
- Decrypts text back to original
- Supports uppercase and lowercase letters

## 🛠️ Tech Used
- Python

## ▶️ How to Run
1. Run the Python file
2. Enter message
3. Enter shift value
